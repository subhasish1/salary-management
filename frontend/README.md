# Frontend – Salary Management System

## 📌 Overview

This is the frontend application for the Salary Management System, designed for HR managers to manage employees and view salary insights through an intuitive user interface.

The application interacts with a Rails API backend to perform CRUD operations and fetch analytical data.

---

## 🎯 Goals

* Provide a clean and intuitive UI for employee management
* Enable seamless interaction with backend APIs
* Display salary insights in a clear and meaningful way
* Maintain a scalable and maintainable frontend architecture

---

## ⚙️ Tech Stack

* React (with Vite)
* JavaScript

---

## 🧠 Design Approach

### 1. Component-Based Architecture

* UI is broken down into reusable components
* Separation between presentation and logic

---

### 2. API-Driven Design

* All data is fetched from backend APIs
* No business logic is tightly coupled with UI

---

### 3. Simplicity & Clarity

* Minimal dependencies
* Focus on readability and maintainability

---

## 📦 Core Features

### Employee Management

* View list of employees
* Add a new employee
* Edit employee details
* Deactivate employee

---

### Salary Insights

* Display average, minimum, and maximum salary by country
* Show average salary by job title within a country

---
